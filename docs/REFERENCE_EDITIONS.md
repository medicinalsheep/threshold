# Reference editions (Phase R — v5.4)

Optional **CC0 reference content** so new creators can practice GLTF insert, textures, sounds, and the **9-step export walkthrough** without sourcing assets first.

---

## Editions

| Edition | Slug | Status | Contents |
|---------|------|--------|----------|
| Threshold Core | `threshold` | Default | Procedural starter scene only |
| **Reference Lite** | `threshold-ref-lite` | **Active** | 3× Kenney GLB (2 vehicles + track) + colormap PNG |

Config: `config/reference-editions.json`

---

## Quick start

```bash
npm run reference:fetch    # download Kenney CC0 assets (first time / refresh)
npm run reference:sync     # copy → import/, textures/, public/bundle/
npm run dev
```

Lobby → **REFERENCE LITE (CC0) →** — spawns vehicles on the starter platform.

Then try **MORE → EXPORT** — **SCENE** step shows real GLTF counts; **CREDITS** lists Kenney CC0.

## Legal policy

**We ship only redistributable assets** (CC0, explicit open licenses).

| OK | Not OK |
|----|--------|
| Kenney, Quaternius, Poly Haven, CC0 Freesound | Ripped Fallout / BF / COD game files |
| Debagged packs with **written CC0/redistribution license** | Typical FiveM vehicle packs without license |
| Your original GIMP/Blender work | Commercial game meshes/audio |

Old-game mod communities are **workflow inspiration** (folder layout, vehicle lists, footstep taxonomy) — see `reference/SOURCES.md`.
