# Controls reference (FiveM-style defaults)

**Version:** 6.5.0 · Rebind in toolbar → **KEYS**

Layout mirrors common **FiveM / GTA** servers: LMB shoot, RMB aim, F vehicle, grouped bindings menu.

---

## On foot (walk mode)

| Action | Default | FiveM parallel |
|--------|---------|----------------|
| Move | WASD | WASD |
| Sprint | Shift (hold) | Shift |
| Jump | Space | Space |
| Crouch | Ctrl (hold) | Ctrl |
| Stealth walk | Alt (hold) | — |
| Interact | E | E |
| Enter vehicle | F | F |
| Fire | **LMB** · G | LMB |
| Aim (ADS) | **RMB** (hold) | RMB |
| Reload | R | R |
| Melee | B | B |
| Holster | Z | — |
| Emote / hands up | X | X |
| Flashlight | L | — |
| Look behind | O | C (in vehicle) |
| FPS / TPS | V | V |
| Third Eye | M | — |
| Voice PTT | N (hold) | N (common on RP servers) |
| Walk / fly toggle | Y | — |
| Session panel | Tab | Tab scoreboard |
| Keys menu | ` | — |

**Mouse:** click canvas in PLAY to capture pointer (aim). **Esc** releases.

---

## Fly / edit

| Action | Default |
|--------|---------|
| Move | WASD + Q up / C down |
| Camera | Mouse orbit (EDIT) or R-stick |
| Walk toggle | Y |

---

## Gamepad (GTA layout)

| Stick / button | Action |
|----------------|--------|
| L-stick | Move |
| R-stick | Camera / aim |
| A / Cross | Jump |
| B / Circle | Melee |
| X / Square | Interact |
| Y / Triangle | Walk / fly · vehicle |
| LT / L2 | Aim (hold) |
| RT / R2 | Fire |
| LB / L1 | Reload |
| L3 | Sprint |
| D-pad Up | Third Eye |
| D-pad Down | FPS / TPS |

---

## Host only

| Action | Default |
|--------|---------|
| Pause (EDIT) | P |

Host keyboard + gamepad profiles sync to guests; guests override in **Guest** profile.
