# iOS platform (generated)

The Capacitor iOS project is **generated locally** and gitignored. Do not edit this folder in git — regenerate on your machine:

```bash
npm run init:native -- --ios
# or
npm run package:ios
```

Open in Xcode (macOS only):

```bash
npm run cap:open:ios
```

See [docs/NATIVE_SHELLS.md](../docs/NATIVE_SHELLS.md) for TestFlight and App Store steps.
